# Sample workspace

This committed workspace gives the desktop app a deterministic local data set for development and demos.
It is the default example workspace used by the native layer when no explicit workspace path is provided.

## Structure

- `workspace.json` — workspace-level metadata
- `targets/` — sample target records
- `runs/` — sample run records
- `run-results/` — stored result envelopes used by local UI flows
- `.ffhn-desktop/app-state.json` — persisted desktop-local sample state
